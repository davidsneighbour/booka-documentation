parseSortString()

Crud13.php : 679

Prepares a sortstring based on a syntaxed string

protected static parseSortString([string $sortstring = '' ]) : array<string|int, mixed>

The returned array can be used by ::prepareSQLSortString() to create the SQL needed for the WHERE part of the query.

Example: query string contains ?sort=-date,title so the function is called as OBJECT::prepareSortParameters($get['sort']) and returns an array array('date' => 'DESC', 'title' => 'ASC')
