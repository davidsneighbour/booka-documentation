index()

StaticIndex.php : 195

Retrieve a filtered list of objects

public static index([array<string|int, mixed> $aParams = [] ]) : array<string|int, mixed>

The values in $params define what filtered index set to return.

Limiting Query

  • limit - count of items to be returned
  • offset - start with item $offset and return $limit items
  • page - start with $page ($offset = ($page-1) * $limit)

$page overrides $offset and if neither is given or $limit is -1 then all items will be returned.

Sorting

  • sort names of the sort columns

minus indicates descending sorting, no minus indicates ascending sorting

If no sorting is defined then each class has it's own sorting setup. Classes without sort setup will return sorted by identifier.

Searching

  • q search query
  • qfields fields to search in

if no qfields are given then search in all title field and titlestring columns.

Fields

  • fields the fields to return
  • expand which fields to expand

If fields is not set then return all columns defined the classes fields array. If no expand is given, then return flat array. With expand set replace fields with their objects.

Note: if fields is not set and the class table has a is_deleted column then objects with this column set to 1 will not be returned. Add explicitly the is_deleted column to retrieve all items.

These fields will be filtered by the fields array of the static class. If the field does not exists in that array, it will be ignored.

Where parameter

  • where a string containing triplets to be used in WHERE requests. Parameter 1 is the column, parameter 2 is the comparison, parameter 3 is the value to be filtered by.

This parameter is only useful if we have filters like date >= today() and such.

Samples:

  • date|>=|today(),date|<=|today()+days(15)
  • id|IN|(1,2,3,4,5)

If you have comparison with = see the section _all other fields

These fields will be filtered by the fields array of the static class. If the field does not exists in that array, it will be ignored.

wherestring

can contain a string that will be added at the end of all other where rules without combining word (like AND, OR and such). Use this for more complicated setups.

Caching

  • cache if the cache parameter is set to a booleanizable value 1 then the request will be cached
  • cachetime if not set it's cached for 3600 seconds (1h)

Associative Arrays

  • assoc if set to a booleanizable value 1 then return associative array containing the idfield in the key

all other fields

all other fields will be added as simple WHERE rule. (like fieldname=value)

These fields will be filtered by the fields array of the static class. If the field does not exists in that array, it will be ignored.
