$requires

AccessControl.php : 54

Contains a required user role.

public static string $requires = 'public'

Add it as parameter to the API method like

@class AccessControl owner

This implies that the user needs to be authorized as this role and the @access phpdoc on the method should be set to protected.

This check is overriding the value of static::$right.
